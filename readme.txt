Instalar dependencias de Python:
    pip install -r requirements.txt
Ejecutar como_root.sql (sólo es necesario una vez):
    mysql < como_root.sql
Ejecutar menu.py:
    python menu.py
Escoger primera opción y ejecutar como_root.sql en otra terminal como root (Cada vez que se ejecute la opción 1 es necesario ejecutar como_root de nuevo):
    mysql < como_root.sql
Escoger opciones según se desee
